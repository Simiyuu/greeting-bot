# Simple Greeting Bot 

A simple JavaScript application that generates a greeting message. This project demonstrates how to run JavaScript code using Node.js directly in the terminal.

##  Features

* Runs in the command line/terminal.
* Built with pure JavaScript.
* Lightweight and fast.

## Prerequisites

To run this project, you need to have **Node.js** installed on your computer.
* [Download Node.js](https://nodejs.org/)

##  Installation

1.  Clone the repository to your local machine:
    ```bash
    git clone [https://github.com/YourUsername/simple-greeting-app.git](https://github.com/Simiyuu/greeting-bot.git)
    ```

2.  Navigate to the project directory:
    ```bash
    cd simple-greeting-app
    ```

##  How to Run

Execute the script using the following command in your terminal:

```bash
node script.js
